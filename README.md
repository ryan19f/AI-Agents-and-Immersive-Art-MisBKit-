# AI Agents and Immersive Art with MisBKit

> Exploring autonomous AI agents, immersive digital art, and real-time communication systems in Unity.
> A Unity-based research project exploring autonomous AI agents, immersive digital art, and real-time communication using ML-Agents, MisBKit, WebSockets, and NDI technologies.

## Overview

This project investigates how AI-driven agents can interact within immersive digital environments. The system combines Unity, machine learning agents, and communication frameworks to create interactive artistic experiences and experimental storytelling environments.

## Technologies

- Unity
- C#
- Unity ML-Agents
- MisBKit
- Native WebSocket
- Klak NDI

## Features

- Autonomous crawler agents
- Real-time communication through WebSockets
- MisBKit integration
- NDI media streaming support
- Interactive immersive environments
